      function isContract(address account) internal view returns (bool)

      Returns whether the target address is a contract, This function will return false if invoked during the constructor of a contract, as the code is not actually created until after the constructor finishes.

      Parameters:
      account - address of the account to check
      Returns:
      whether the target address is a contract
